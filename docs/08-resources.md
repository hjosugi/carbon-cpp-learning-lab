## Freshness checklist

教材を更新する時は次を確認します。

1. latest nightly tagとasset名
2. READMEのgetting-started command
3. roadmapのlatest objective
4. merged proposalsのsyntax changes
5. design docs内の`TODO`/`Future work`
6. smoke labsのcompile/link/run
7. deprecated Explorer interpreter向け記事を参照していないこと
